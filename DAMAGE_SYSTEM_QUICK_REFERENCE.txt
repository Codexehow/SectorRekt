================================================================================
                        DAMAGE SOURCES IN GAME
================================================================================

1. WALL COLLISION
   When:       Player slides against TileMapLayer or StaticBody2D
   Damage:     0.5 HP per second (while sliding)
   Triggered:  Automatic in _physics_process()
   
2. ENEMY COLLISION
   When:       Enemy touches DamageZone (Area2D radius 30)
   Damage:     25 HP per touch
   Triggered:  _on_body_entered() → body_entered signal
   Effect:     Also causes camera shake (intensity 5)

3. CUSTOM
   Usage:      player.take_damage(amount, "source_name")
   Example:    player.take_damage(50.0, "boss_attack")

================================================================================
                           SIGNALS (FOR UI)
================================================================================

player_damaged(hull: float, shield: float)
  → Emitted after EVERY damage event
  → Use to update health bars in real-time
  
player_died
  → Emitted when hull reaches 0
  → Use to show game-over screen

Example listener:
    func _ready():
        player.player_damaged.connect(_on_damage)
    
    func _on_damage(hull, shield):
        health_bar.value = hull
        shield_bar.value = shield
