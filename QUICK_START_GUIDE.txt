1. CONTROLS
═══════════════════════════════════════════════════════════════════════════════════

GENERATE RESOURCES:
  Q Key              → Generate 25 CPU cycles
  Right Mouse Button → Generate 25 CPU cycles
  
ATTACK:
  Left Mouse Button  → Fire Thunderbolt (requires 30+ weapon charge)
  
MOVEMENT:
  W / Up Arrow       → Accelerate forward
  S / Down Arrow     → Decelerate
  A / Left Arrow     → Turn left
  D / Right Arrow    → Turn right
  Mouse Position     → Aim weapon
  1-5 Keys           → Set speed presets
  
SPECIAL:
  B Key              → Blink Drive (requires 100 blink charge)
  G Key              → Toggle Corruption Mode

2. RESOURCE SYSTEM
═══════════════════════════════════════════════════════════════════════════════════

WHAT IS CPU?
  → Your primary resource pool
  → Max capacity: 100 cycles
  → Automatically distributes to 5 systems
  
HOW TO GENERATE CPU:
  → Click Q or Right Mouse Button
  → Each click gives +25 CPU
  → Stacks up to 100 max
  → Decay: -15 cycles/second when idle
  
DISTRIBUTION (Automatic):
  ┌─────────────────────────────────┐
  │ CPU Generation → Distribution   │
  ├─────────────────────────────────┤
  │ 50% → Weapon Charge (orange)    │
  │ 30% → Shield Charge (blue)      │
  │ 10% → Movement Bonus            │
  │ 10% → Life Support              │
  │ 10% → Blink Charge (magenta)    │
  └─────────────────────────────────┘

6. KEY MECHANICS
═══════════════════════════════════════════════════════════════════════════════════

CLICK-BASED GENERATION
  → NOT a holding system - it's click & release
  → Each click is instant +25 CPU
  → More tactical than holding down a button
  → Frees up your button from holding while dodging
  
CPU DECAY
  → When you stop generating CPU, it slowly drains
  → Rate: -15 cycles/second
  → Full 100 CPU takes ~6.7 seconds to drain completely
  → Encourages active resource management
  
AUTOMATIC DISTRIBUTION
  → No manual allocation needed
  → All systems feed simultaneously
  → Weapon gets the most (50%)
  → Everything scales together
  
MOVEMENT BONUS
  → 10% of CPU generation increases speed
  → More CPU = slightly faster movement
  → Never stops you from moving

9. TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════════

"My weapon won't fire!"
  → Check weapon bar - need at least 30 charge
  → Make sure you're clicking Q first to generate CPU
  → Wait for weapon bar to fill up (orange color)
  
"CPU decays too fast!"
  → That's intentional - forces you to play actively
  → Click Q more frequently to maintain high CPU
  → Plan your attacks with the decay in mind
  
"Blink didn't work!"
  → Blink bar must be FULL (100, not 99)
  → Takes several clicks to charge fully
  → Press B key when bar is completely filled
  
"Where are the UI bars?"
  → Check top-left corner of screen
  → Should show 4 bars: CPU, Weapon, Shield, Blink
  → If missing, restart the scene
