QUICK START
═══════════════════════════════════════════════════════════════════════════

1. Open Godot project
2. Open scene: res://world.tscn
3. Click Play (or F5)
4. Move player toward walls
5. Watch for cyan glimmer when shields take damage
6. Check UI top-left for Shield and Hull bars


WHAT TO LOOK FOR
═══════════════════════════════════════════════════════════════════════════

✓ CYAN GLIMMER EFFECT
  Location: Over the yellow spider tank
  When: When shields absorb damage
  Duration: 0.15 seconds
  Animation: Flickers 8 times while fading
  Expected: Bright blue/green flash

✓ SHIELD BAR
  Location: Top-left UI panel
  Color: Cyan
  Changes: Decreases when taking damage
  Max: 100/100

✓ HULL BAR
  Location: Top-left UI panel (below Shield bar)
  Color: Red (normal), Bright Red (critical)
  Changes: Only decreases when shields are depleted
  Max: 100/100

✓ CONSOLE OUTPUT
  Open: View → Output Console
  Look for:
    - "Shield absorbed X damage from wall"
    - "Hull took X damage from wall"


TESTING SCENARIOS
═══════════════════════════════════════════════════════════════════════════

SCENARIO 1: Wall Collision with Shields
─────────────────────────────────────────
1. Start game
2. Walk slowly toward a wall
3. Watch Shield bar decrease
4. See cyan glimmer appear
5. Hull bar stays at 100

Expected Result: ✓ PASS
  • Cyan flash appears
  • Shield bar decreases
  • Hull bar unchanged
  • Console: "Shield absorbed X damage from wall"


SCENARIO 2: Shield Depletion
──────────────────────────────
1. Continue into wall
2. Watch shield reach 0
3. Continue taking damage
4. Hull bar starts decreasing
5. NO MORE cyan glimmer

Expected Result: ✓ PASS
  • Shield bar at 0/100
  • No cyan flash (shields gone)
  • Hull bar decreasing
  • Console: "Hull took X damage from wall"


SCENARIO 3: Enemy Contact
──────────────────────────
1. Run away from wall
2. Let shields regenerate (if enabled)
   OR reset level with reload
3. Find an enemy in the level
4. Walk into enemy
5. Watch for cyan glimmer
6. Shield bar decreases by 25

Expected Result: ✓ PASS
  • Cyan flash appears
  • Shield absorbs 25 damage
  • Hull unchanged
  • Console: "Shield absorbed 25 damage from enemy"


SCENARIO 4: Critical Hull State
────────────────────────────────
1. Let shields break (scenario 2)
2. Take more wall damage
3. Damage hull to ≤ 25%
4. Watch Hull label turn bright red
5. Continue damaging

Expected Result: ✓ PASS
  • Hull label color: Normal red → Bright red
  • Stays bright red while ≤ 25%
  • Returns to normal red if > 25%


SCENARIO 5: Game Over
──────────────────────
1. Continue taking damage
2. Let hull reach 0
3. Game should end

Expected Result: ✓ PASS
  • Scene shows "SYSTEM CORRUPTED"
  • Scene reloads after delay
  • Game restarts


CONTROLS
═══════════════════════════════════════════════════════════════════════════

Arrow Keys / WASD:   Move tank
Mouse:               Aim and fire (optional)
Q or Right Mouse:    Generate CPU (optional)

For testing damage:
  • Use ARROW KEYS to move toward walls
  • Let collisions deal damage naturally


TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════

❌ No cyan glimmer showing?
   → Check if shields > 0
   → Try walking into wall slower
   → Check console for damage messages

❌ Hull bar not visible?
   → Scroll up in UI panel
   → Check if HullLabel and HullBar in scene

❌ Hull bar not updating?
   → Deplete shields first
   → Then damage hull
   → Check console for "Hull took X damage"

❌ Game crashes?
   → Check Output console for errors
   → Check that all files exist:
     - res://player/impact_glimmer.gd
     - res://player/player.tscn (with ImpactGlimmer)
     - res://ui/cpu_hud.gd (updated)

❌ Can't see enemies?
   → Enemies spawn randomly in rooms
   → Navigate through level
   → Or just test with walls


PERFORMANCE CHECK
═══════════════════════════════════════════════════════════════════════════

Check Memory:
  • No growing memory usage

  • Reuses existing nodes
