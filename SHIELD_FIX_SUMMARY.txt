================================================================================
TECHNICAL DETAILS
================================================================================

DAMAGE FLOW:
  Incoming Damage → Shields Active? 
                   ├─ YES → Shield absorbs → Reduced Shield ✓
                   └─ NO → Goes to Hull → Reduced Hull ✓

SHIELD REGENERATION:
  Take Damage → Timer Reset to 0
               ↓
              Wait 3 seconds (timer increments)
               ↓
              Shield < Max? → Yes → Regen at 20 pts/sec
                               ↓
                           Shields Full → Stop Regen
                               ↓
                           Ready for next damage

VISUAL EFFECTS:
  Shield Impact → ImpactGlimmer Triggers
                 ├─ Cyan Color
                 ├─ 0.15 sec duration
                 ├─ Flicker fade-out
                 └─ Show only when shields absorb
