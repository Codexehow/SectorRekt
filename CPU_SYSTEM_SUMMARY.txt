CORE MECHANIC: Active Resource Management

   └─ Automatically distributed across 5 critical subsystems

┌──────────────────────────────────────────────────────────────────────────────┐
│ ⚡ CPU GENERATION & DISTRIBUTION                                              │
└──────────────────────────────────────────────────────────────────────────────┘

GENERATION:
  Input: Q Key OR Right Mouse Button (hold to generate)
  Rate: +35 cycles per second
  Max: 100 cycles
  Decay: -15 cycles per second when idle
  
AUTOMATIC DISTRIBUTION:
  50% → Weapon (Thunderbolt)           🔴 Fire projectiles at enemies
  30% → Shields                         🟦 Block incoming damage
  10% → Movement                        🟢 Speed boost (+10% max)
  10% → Life Support                    🟨 Reserved for health system
  10% → Blink Drive                     🟪 Teleportation ability

RESULT: Balanced resource management forces difficult tactical decisions

   ├─ Cost: Full charge (depletes to 0)
   ├─ Effect: Teleports player 150 pixels forward
   ├─ Feature: Can pass through walls
   ├─ Direction: Forward along player rotation
   └─ Status: ✅ Fully Implemented & Working

   ├─ Input: Automatic (continuous from MOVEMENT_ALLOC)
   ├─ Requirement: CPU generation active
   ├─ Effect: Up to +10% speed boost

│ 📡 SIGNALS FOR UI INTEGRATION                                                │
└──────────────────────────────────────────────────────────────────────────────┘

SIGNAL DEFINITION:
  signal cpu_updated(current: float, weapon: float, shield: float, blink: float)

EMISSION:
  • Every frame in _physics_process()
  • Guaranteed to emit continuously during gameplay
  • All four float values (0.0 to 100.0)

USAGE EXAMPLE:
  player.cpu_updated.connect(func(current, weapon, shield, blink):
      cpu_bar.value = current / 100.0
      weapon_bar.value = weapon / 100.0
      shield_bar.value = shield / 100.0
      blink_bar.value = blink / 100.0
  )

UI SETUP:
  • Create 4 ProgressBar nodes
  • Connect to cpu_updated signal
  • Update bars every frame
  • Display real-time feedback to player
