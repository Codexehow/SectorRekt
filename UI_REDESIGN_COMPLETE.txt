================================================================================
                            HOW TO USE IT
================================================================================

IN-GAME:
  1. Launch the game
  2. See Resource Panel in top-left corner
  3. See Controls Panel in bottom-left corner
  4. Press Home key to toggle controls (show/hide)
  5. Watch CPU bar color change as you play
  6. Center screen remains completely clear for gameplay

CUSTOMIZATION:
  1. Toggle key can be changed (currently 'Home' or 'ui_home')
  2. Colors can be customized in script if needed
  3. Panel sizes can be adjusted (Control offset properties)
  4. UI transparency can be modified (StyleBox colors)

TROUBLESHOOTING:
  - If UI looks pixelated: Check integer scaling settings
  - If bars don't update: Verify player is in "player" group
  - If toggle doesn't work: Check input map for "ui_home" action
  - If colors are wrong: Verify monitor color space is sRGB
