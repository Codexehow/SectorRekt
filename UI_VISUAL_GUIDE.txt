COLOR REFERENCE GUIDE
================================================================================

Primary Accent (Neon Green):
  RGB: (0, 255, 0) or Color(0, 1, 0)
  Hex: #00FF00
  Uses: Borders, primary text, bar highlights
  
  Example: 
  ╔════════════════╗
  ║ Border and     ║  ← Bright green (0, 1, 0)
  ║ CPU Text       ║
  ╚════════════════╝

Light Green (Secondary):
  RGB: (179, 255, 179) or Color(0.7, 1, 0.7)
  Hex: #B3FFB3
  Uses: Value text, secondary text, subtle highlights

Dark Background:
  RGB: (13, 13, 20) or Color(0.05, 0.05, 0.08)
  Hex: #0D0D14
  Uses: Panel background, creates contrast

CPU BAR DYNAMIC COLORING (Most Important Feature!)
================================================================================

     0% ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ 100%
        │         │         │         │         │
     RED(1,0,0) ORANGE(1,0.8,0) YELLOW-GREEN BRIGHT-GREEN(0,1,0)
        │         │         │         │         │
     Critical   Low      Good      Optimal   Full
     Recharge!  Caution   Continue  Power!    Max
       <20%      20-50%   50-80%   80-100%   ≥100%

Visual Feedback Loop:
1. Player uses energy → CPU drops → bar color changes
2. Visual color change indicates status
3. Player can react instantly without reading numbers
4. Encourages resource management


SCREEN LAYOUT MATH
================================================================================

Typical Resolution: 1920x1080

Resource Panel Footprint:
  Position: (0, 0)
  Size: 300x250px
  Screen Coverage: 300/1920 × 250/1080 = 4.3%

Controls Panel Footprint (when visible):
  Position: (0, 960) [screen_height - 120]
  Size: 300x120px
  Screen Coverage: 300/1920 × 120/1080 = 2.1%

Total UI Coverage: ~6.4% when both visible
Center Gameplay Area: ~93.6% completely clear

Advantage: UI only occupies corner regions
Player's focus: Center 80% of screen


TEXT SIZING HIERARCHY
================================================================================

16px ← CPU Label (Primary - LARGEST)
      "CPU"

24px ← CPU Bar height (Visual prominence)

12px ← Secondary labels + CPU value
      "Weapon"   "Shield"   "Blink"
      "42/100"

11px ← Controls title
      "[CONTROLS] (Press Home)"

10px ← Instructions (Smallest - least important)
      "Q/RMB: CPU | LMB: Fire | B: Blink | A/D: Turn | W/S: Speed | G: Corruption"


INTERACTIVE ELEMENTS
================================================================================

Controls Panel Toggle:
  Input: Home key (customizable)
  Effect: Show/Hide bottom-left panel
  State Management:
    Visible = true  → Panel shown, takes 120px height
    Visible = false → Panel hidden, reclaims space

Bar Updates:
  Input: Player signals (cpu_updated)
  Update Rate: ~60 FPS
  Components Updated:
    - Bar value (0-100)
    - Bar color (dynamic based on level)
    - Text value ("X/100")
